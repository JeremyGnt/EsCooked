> 🎓 Réalisé dans le cadre d’un projet scolaire avec l’**ECE Lyon**.  
> 📜 Les droits associés au projet appartiennent à l’**ECE Lyon**.

---

## 🚀 À propos du projet

**EsCooked** est un projet logiciel bas niveau en C, réalisé en équipe, mettant l’accent sur :

- la qualité de code en C (modularité, lisibilité, robustesse),
- l’organisation d’un dépôt collaboratif,
- la mise en place d’une chaîne de compilation propre avec CMake,
- la résolution de problèmes techniques en contexte d’équipe.

## ⚙️ Installation & compilation

### Prérequis
- Compilateur C (GCC / Clang / MSVC)
- CMake
- Make ou Ninja

### Build

```bash
git clone https://github.com/JeremyGnt/EsCooked.git
cd EsCooked

cmake -S . -B build
cmake --build build
```

### Exécution

```bash
./build/EsCooked
```

> Le nom exact du binaire peut varier selon la configuration CMake.
